# Novitec Portal

Portal ejecutivo web para consumir la API analitica de Novitec.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Aceternity UI
- TanStack Query
- TanStack Table
- Apache ECharts

## Variables de entorno

- `VITE_API_BASE_URL`
- `VITE_API_TOKEN`

## Scripts

```powershell
pnpm dev
pnpm build
pnpm check
```
